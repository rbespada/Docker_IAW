========================================
FLUJO DE UNA PETICIÓN HTTP (Ejemplo)
========================================

USUARIO: "Click en Agregar al Carrito"
    ↓
NAVEGADOR: fetch('/api/carrito', POST)
    ↓
TLS NEGOTIATION: Handshake HTTPS
    ↓
NGINX (Puerto 80/443):
    • Recibe petición
    • Identifica: /api/ → dinámico
    • Reenvía a app:3000
    ↓
NODE.JS + EXPRESS:
    • Valida sesión del usuario
    • Consulta: SELECT * FROM productos
    • Encuentra producto ID=1
    • Añade al carrito
    ↓
SQLite:
    • Retorna datos del producto
    ↓
RESPUESTA HTTP:
    Code: 200 OK
    Content-Type: application/json
    Content-Encoding: gzip
    
    {"mensaje": "Añadido", "carrito": [...]}
    ↓
NAVEGADOR:
    • Descomprime JSON
    • Ejecuta JavaScript
    • Actualiza carrito visual
    ↓
USUARIO: Ve "✓ Producto añadido"
